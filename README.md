# Personal Portfolio Website

This is a personal portfolio website built with Next.js, Tailwind CSS, and TypeScript. It showcases my work, provides an AI-powered portfolio review feature, and includes a contact form for inquiries.

## Features

- **Responsive Design:** The website is fully responsive and works on all devices.
- **Project Showcase:** Display your projects with images, descriptions, tags, and links to live demos and GitHub repositories.
- **AI Portfolio Review:** Get AI-powered feedback on your portfolio content, such as project descriptions or your bio.
- **Contact Form:** A functional contact form to receive messages from visitors.
- **Dark Mode:** The website supports a dark mode theme.

## Getting Started

### Prerequisites

- Node.js (v18.0.0 or later)
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repository
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

To run the development server, use the following command:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Running Tests

To run the tests, use the following command:

```bash
npm run test
```
