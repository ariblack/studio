# **App Name**: AzurePrint

## Core Features:

- Profile Showcase: Visually appealing display of your projects and professional background.
- Project Gallery: Interactive gallery to showcase your design and development projects.
- AI-Powered Portfolio Review Tool: Use an AI tool to review a section of the portfolio against best practices, for feedback and actionable steps.
- Contact Form: Simple contact form for visitors to get in touch.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) for a clean, professional feel.
- Background color: Off-white (#F5F5F5) for a soft, unobtrusive backdrop.
- Accent color: A darker blue (#4682B4) to highlight key elements and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look. It can be used for headlines and body text.
- Simple, minimalist icons to enhance navigation and visual interest.
- Clean and spacious layout with a focus on readability and visual hierarchy.
- Subtle transitions and hover effects to add interactivity.
